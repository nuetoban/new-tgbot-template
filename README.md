# new-bot-template

This repo contains sh-script which I use to create a new Telegram bot from scratch.
After running the script you will have the following layout:

```
.
├── Dockerfile
├── .env
├── .env.example
├── .git
├── .gitignore
├── main.py
├── Makefile
├── requirements.txt
├── tag.txt
├── tests
└── venv
```

## Usage
```./new-bot.sh your-bot-name```
